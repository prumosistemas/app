# Legado

Esta pasta guarda relatorios e anotacoes antigas que nao fazem parte do caminho ativo de deploy.

Os HTMLs publicados pelo Netlify continuam na raiz do repositorio. Nao mova `index.html`, `login.html`, `admin.html`, `iss-fortaleza.html`, `master.html` ou `master-company.html` para ca sem ajustar a configuracao de deploy.

## Organizacao

- `relatorios/`: checklists e relatorios de entrega/auditoria anteriores.
- `docs/`: notas tecnicas antigas ou tarefas historicas.
